Problema:


Mecanismo para conceder descontos aos clientes;
Divulgação de promoções específicas de cada produto;
Dificuldade de verificar o preço dos produtos quando não há a etiqueta do preço;

 
Solução:

Criação de um aplicativo, onde o cliente, utilizando um smartphone, ao ler o código de barras do produto
pode receber algum desconto ou saber se existe algum tipo de promoção para aquele produto.
Ex.: O Cliente faz a leitura do código de barras do produto X e este produto X está com desconto de 50% na segunda unidade ou
na compra de 3 unidades compre por R$ 0,01 o produto Y. 

Atores: Gerente, clientes.

Requisitos Funcionais

-Mostrará os valores dos produtos em tempo real e se há possíveis descontos;
-Fará a soma dos produtos para conferir com o total do caixa;
-Mostrara se há descontos em produtos similares;

Requisitos não funcionais

-Utilizara JavaScript 
-Utilizara banco de dados MYSQL
-Atualização constante com o supermercado.
